#  Chaos Kitchen (unity)

A simple **Chaos Kitchen clone** developed using **Unity** for learning and practice purposes.

---

##  Description

This project is a recreation of the **Chaos kitchen** game, built with **Unity and C#**.  
The main objective of this project is to practice fundamental game development concepts such as physics-based movement, raycast, collision detection, scoring systems, and scene management.

This project was created as part of my learning journey as a **Game Development student**.

---

##  Gameplay

- Using WASD to move
-  Make dishes to serve customer with recipes on the right corner
- Score increases when the dish is delivered successfully

---

##  Features

- Physics-based chef movement with unity gameInput system
- Score tracking system
- Recipe spawn system
- Game over and restart logic  
- Simple UI (score, restart button)

---

##  Technologies Used

- **Game Engine:** Unity  
- **Programming Language:** C#  
- **Version Control:** Git & GitHub
